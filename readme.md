# 在Spring boot下测试Druid

主页   http://localhost:8080/  
  
测试  Druid 的示例  
  
Druid的监视连接：      
  http://localhost:8080/druid/  
  
pom.xml中添加

    <!-- druid   添加 -->  
	<dependency>  
		<groupId>com.alibaba</groupId>  
		<artifactId>druid-spring-boot-starter</artifactId>  
		<version>1.1.2</version>  
	</dependency>
